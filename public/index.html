Math Machine (beta)
Math Machine? What is Math Machine?
Math Machine is a web service for doing simple math operations using clean urls.
And how do I use this thing?
All you have to do is prefix the math operator to its arguments. Like this:
/add/2/1
Is there a list of operations?
-6 ................. /neg/6
|9| ................ /abs/9
2 + 1 .............. /add/2/1
9 - 3 .............. /sub/9/3
5 * 7 .............. /mul/5/7
8 / 4 .............. /div/8/4
2 ^ 3 .............. /pow/2/3
sqrt(9,2) .......... /sqrt/9/2
